Calendar : Add a new events

Vous devez aller sur l'onglet "Events", situé dans le dashboard à droite de votre écran. Une page va s'ouvrir avec tous les évènements publiés sur votre site.

Pour ajouter un nouvel évènement, cliquer sur "Add New". Vous arrivez sur une page vous permettant de créer votre évènement.
Dans la barre de saisie, tapez l'intitulé de l'article. Dans le second bloc de saisie texte, vous avez la possibilité d'ajouter le descritif de votre choix ou de ne rien stipuler. Les champs suivants vont permettrent d'ajouter des précisions à votre évènement.

Time and date

Dans cette catégorie, sélectionner votre date de début et de fin ainsi que les horaires. Si l'évènement a lieu sur la journée entière, cochez la case située en dessous.

Location

Venue: sélectionner une localisation existante ou en créer une nouvelle. Décocher les 2 cases situées en dessous.

Organizers

Rentrer un nouvel organisateur dans la barre de saisie ou sélectionner un organisateur existant.

Event Webstite or event cost

Ajouter l'adresse web du site internet relatif à l'évènement et/ou son prix. Si l'évènement ne comporte aucunes de ces particularités, il vous est tout à fait possible de laisser des champs vides. N'oubliez pas de preview et publish si cela vous convient.

For edit or delete events

Sur la page initiale "Events", survolez les évènements avec la sourie. A cet instant, plusieurs options apparaissent comme Edit ou Trash.

Training

Il est nécessaire d'aller dans l'onglet page sur le dashboard et éditer la page "Training". Au survol du texte, un encart vert apparaît. Cliquer sur "Edit Text Block" et modifier directement le texte dans le bloc de saisie. Ne rien ajouter dans les autres champs. N'oubliez pas de "Save changes"!

Pour modifier la vidéo déjà présente, suivez la même procédure que pour le bloc texte sauf qui'il vous suffira juste d'ajouter le lien vers la vidéo Viméo souhaitée.

Pour ajouter ou supprimer du contenu sur la page

Pour ajouter un bloc (texte, image ou vidéo), cliquer sur le "+" situé tout en bas de la page ou en haut dans l'onglet bleu. Au survol, il sera indiqué "Add element". Ensuite sélectionnez soit "Text block" pour du texte, soit "Single image" pour une photo dans les onglets "All" ou "Content". Pour une vidéo, allez dans "QuCreative Elements" et sélectionnez "QuCreative - Video Player". Voir les images ci-dessous :

Pour supprimer un bloc de contenu, cliquer sur la croix en haut à droite du bloc en question.

Page About

Pour modifier la photo ou le texte de l'encart descriptif, allez sur la page about et survolez le bloc en question. Sélectionnez ensuite "Edit QuCreative - Image Box" et modifiez les informations dans l'encart qui s'ouvre. Vous avez la possibilité d'ajouter des images (le + vert) ou supprimer la photo présente pour la remplacer (la croix rouge sur la photo). Puis, modifier le texte tout en bas. N'oubliez pas de "Save changes".

Collaborators

Dupliquer le dernier "QuCreative - Team member" et éditer ce dernier. Dans l'encart qui s'ouvre, modifier "Avatar" et y sélectionner la photo correspondante, "first name", "last name" et "position". N'oubliez pas de "Save changes"

Page Contact : change the E-mail address

Allez sur la page "Contact" puis survolez le champ afin de pouvoir cliquer sur "Edit". Modifiez l'adresse mail déjà renseignée par la nouvelle souhaitée.
